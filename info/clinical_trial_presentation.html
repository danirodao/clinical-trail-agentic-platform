Agentic Lakehouse Strategy

The LangGraph ReAct loop separates reasoning from execution, with explicit safety guardrails.

# System Prompt Guardrail Example IF query_spans_multiple_trials: ASSERT access_level == "aggregate" DO NOT request individual patient rows. # Tool Selection Strategy Use `search_documents` for protocols. Use `get_patient_demographics` for cohorts. Use `compare_treatment_arms` for efficacy.

Fine Grain Access Control

Three composable layers evaluated per trial, enforcing the Access Level Ceiling.

1️⃣
OpenFGA ReBAC
Trial-level binary access (Aggregate vs Individual).
2️⃣
Governance Ceiling
ABAC constraints (e.g., EU region only) applied per trial.
3️⃣
SQL Cohort Filters
PostgreSQL JSONB criteria compiled into dynamic WHERE clauses.
